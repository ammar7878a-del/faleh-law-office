## 🎯 خطوات النشر السريعة

### 1. إعداد GitHub (5 دقائق)
```bash
1. إنشاء حساب GitHub
2. إنشاء مستودع جديد
3. رفع ملفات المشروع
```

### 2. إعداد قاعدة البيانات (10 دقائق)
```bash
1. إنشاء حساب Supabase
2. إنشاء مشروع جديد
3. نسخ رابط قاعدة البيانات
```

### 3. النشر على Render (15 دقائق)
```bash
1. إنشاء حساب Render
2. ربط مستودع GitHub
3. إضافة متغيرات البيئة
4. نشر المشروع
```
